# :sunflower: lane detection

라즈베리파이를 이용해서 라인 검출하기

# Dependency
- :strawberry: Raspberry pi 3B+
- :car: XYTRON

# ORDER

```
BGR to Gray ->
Canny Edge Detection ->
가우시안 정규화 ->
관심영역 필터링 ->
허프 직선검출 ->
차선 검출 ->
차선 그리기
```
